# Self-Improvement PRD

## Requirement
Run output on console shows how many iterations were needed and how long it took to complete the project. Each agent should output how much time it took them to finish its job. Track and display score trends across iterations (e.g. Review: 60→75→88) so the human can see convergence or stalling.

## Context
This is the ADT codebase itself. Make the requested improvements while maintaining the existing architecture and conventions.

## Acceptance Criteria
- The requested improvement is implemented
- Existing functionality is not broken
- Code follows project conventions
- npm run build succeeds
